# 🔍 GitHub Users Search

A modern, fast, and type-safe GitHub user search application built with React, Vite, and Tailwind CSS. Simply search for a GitHub username and instantly explore detailed user profiles and their activity.

![GitHub Users Search Demo](https://via.placeholder.com/800x400?text=GitHub+User+Search+Demo)

## ✨ Features

- 🔎 **Real-time GitHub User Search** – Search any GitHub username and get instant results
- 👤 **Detailed User Profiles** – View avatar, bio, followers, following, repositories, and more
- 📊 **User Activity Timeline** – See recent user activity using GitHub GraphQL API
- 🎨 **Beautiful UI Components** – Built with shadcn/ui for a polished, accessible interface
- ⚡ **Blazing Fast** – Powered by Vite for near-instant development and build times
- 🛡️ **Type Safe** – Full TypeScript support for better developer experience and fewer bugs

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **React 18** | UI Framework |
| **Vite** | Build tool & Dev Server |
| **TypeScript** | Type Safety |
| **Tailwind CSS** | Styling |
| **shadcn/ui** | Component Library |
| **Apollo Client** | GraphQL Client |
| **GitHub GraphQL API** | Data Source |
